napari 0.4.9¶

We’re happy to announce the release of napari 0.4.9! napari is a fast, interactive, multi-dimensional image viewer for Python. It’s designed for browsing, annotating, and analyzing large multi-dimensional images. It’s built on top of Qt (for the GUI), vispy (for performant GPU-based rendering), and the scientific Python stack (numpy, scipy).

For more information, examples, and documentation, please visit our website: https://github.com/napari/napari

Highlights¶

This release adds a couple nice new features like additional shading modes for our surface layer (#2972) and the ability to copy a screenshot directly to the clipboard (#2721). It also contains a variety of bug fixes and improvements.

New Features¶

  • Add tooltip for labels (#2658)

  • Added copy-to-clipboard functionality (#2721)

  • Add make watch command for hot reload (#2763)

  • Expose alternative shading modes for surfaces (#2792)
